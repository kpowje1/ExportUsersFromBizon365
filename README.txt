С помощью этого кода можно делать выгрузку по вебинарным комнатам с сервиса BIZON365.
На самом сервисе нет массовой выгрузки, если нужно получить данные по посетителям за 30 дней, то нужно в каждую
сессию вебинара заходить отдельно и выгружать в файл. Текущий код решает эту задачу, можно указать ID комнаты и даты (в рамках одного месяца)
с какого по какое число нужно сделать выгрузку.
В коде указан 2021 год, так как в это время использовался. Написан был под конкретную ситуацию, поэтому далее доработки не было.
